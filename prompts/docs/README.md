# Google Docs Prompts

## 阅读与总结

### 文档摘要
```
读取这篇 Google Doc：[链接]
输出：① 核心结论 ② 主要论点（3–5 条）③ 待跟进事项
```

### 对比两个文档
```
对比 [文档 A 链接] 和 [文档 B 链接]，找出主要差异点，用表格呈现。
```

## 撰写与编辑

### 起草新文档
```
帮我创建一份 [类型：会议纪要/项目方案/周报/技术文档]：
主题：[xxx]
受众：[xxx]
关键内容：[xxx]
长度：约 [N] 字
保存到 Drive [文件夹]。
```

### 润色文档
```
读取 [文档链接]，对以下方面进行改进：
① 逻辑结构更清晰 ② 语言更简练 ③ 标题层级规范
输出修改建议，我确认后再更新。
```

### 会议纪要生成
```
根据以下会议记录，生成标准格式的 Google Doc 会议纪要：
- 参会人：[xxx]
- 时间：[xxx]
- 讨论内容：[粘贴原始记录]
保存到 Drive [路径]。
```
