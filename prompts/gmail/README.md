# Gmail Prompts

## 收件箱概览

### 今天邮件清单
```
帮我查看今天收到的所有邮件，按发件人和主题列成清单，标注【需回复 / 知晓即可 / 可忽略】。
```

### 搜索邮件
```
搜索所有来自 [发件人] 关于 [关键词] 的邮件，按时间倒序，摘要每封核心内容（一句话）。
```

### 未读邮件整理
```
扫描所有未读邮件，分三类整理：
1. 需要我回复的
2. 仅需知晓的通知
3. 订阅/广告（可批量忽略）
```

## 撰写与发送

### 起草回复
```
帮我回复这封邮件，语气：[正式/友好/简洁]，核心意思：[xxx]，字数 ≤ 150 字。
草稿生成后请我确认再提交到 Gmail。
```

### 新建邮件
```
帮我起草一封邮件：
收件人：[邮箱]
主题：[主题]
内容要点：[xxx]
语气：[正式/轻松]
```

### 未回复跟进
```
找出我发出后超过 [N] 天还没收到回复的邮件，列出主题、收件人、发送时间，我来决定是否跟进。
```

## 移动场景

### 快速三句摘要
```
把最新 10 封未读邮件每封用一句话摘要，让我在手机上快速判断优先级。
```

### 语音转邮件
```
把以下口语内容整理成正式邮件草稿，保留核心意思，润色语气：
[粘贴语音转写内容]
```
