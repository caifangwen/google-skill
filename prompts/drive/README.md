# Google Drive Prompts

## 文件管理

### 搜索文件
```
在我的 Drive 中搜索关于 [关键词] 的文件，列出文件名、类型、最后修改时间，按相关度排序。
```

### 最近文件概览
```
列出我最近 7 天内修改过的文件，按类型分组（文档/表格/演示/其他）。
```

### 整理建议
```
扫描我 Drive 根目录的文件结构，建议一套更清晰的文件夹分类方案。
```

## 内容操作

### 读取文档内容
```
读取 Drive 中的 [文件名/链接]，总结主要内容，列出关键结论和待办事项。
```

### 保存到 Drive
```
将以下内容保存为 [文件名].md 到我的 Drive [文件夹路径] 中：
[内容]
```

## 移动场景

### 快速文件搜索
```
帮我找上周和 [某人] 共享的文档，按修改时间排列最新的 3 个。
```
