# Google Tasks Prompts

## 查看与管理

### 今日任务列表
```
列出我所有 Tasks 清单中今天到期或逾期的任务，按优先级排序。
```

### 本周任务概览
```
展示本周内到期的所有任务，标注：清单名称、截止日期、是否已完成。
```

## 创建与更新

### 快速添加任务
```
帮我在 [清单名称] 中添加以下任务：
任务名：[xxx]
截止日期：[日期]
备注：[xxx]
```

### 批量添加任务
```
把以下事项批量加入 Tasks 的 [清单名称]，截止日期统一为 [日期]：
- [任务 1]
- [任务 2]
- [任务 3]
```

### 标记完成
```
把 [清单名称] 中的 [任务名] 标记为已完成。
```

## 移动场景

### 今天最重要的 3 件事
```
综合我的 Tasks 和 Calendar，告诉我今天最重要的 3 件事，
一句话说明原因。
```

### 睡前收尾检查
```
今天有哪些任务没完成？是否需要顺延到明天？帮我整理一份简短的收尾清单。
```
