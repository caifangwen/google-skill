# Cross-Tool Integration Workflows

## 工作日开始

### 晨间智能简报 (Calendar + Tasks + Gmail)
```
综合以下内容，给我生成今天的工作简报：
① 今天的日历事件
② 今天到期的 Tasks
③ 昨晚收到的重要邮件
格式：3 个模块，每模块 3–5 条，加粗重点。
```

## 完整会议流程

### 会前准备 (Calendar + Gmail + Drive)
```
我在 [时间] 有关于 [主题] 的会议，请：
1. 确认日历事件详情和参会人
2. 搜索与参会人最近的邮件往来
3. 在 Drive 中查找相关文档
4. 输出一份 200 字的会前背景摘要
```

### 会后整理 (Docs + Tasks + Calendar)
```
根据以下会议记录，请：
1. 生成标准会议纪要并保存到 Drive [路径]
2. 把所有 Action Item 添加到 Tasks [清单名]，设置截止日期
3. 若有后续会议约定，在 Calendar 创建事件
[粘贴会议记录]
```

## 项目管理

### 项目周报 (Calendar + Tasks + Sheets)
```
综合本周的：
① Calendar 事件（会议/里程碑）
② Tasks 完成情况
③ [Sheet 链接] 中的进度数据
生成一份周报草稿，包含：本周完成、下周计划、风险/阻碍。
```

### 项目任务拆解 (Tasks + Calendar + Docs)
```
我有一个项目：[项目名]，截止日期：[日期]
主要交付物：[xxx]
请：
1. 拆解为 5–10 个子任务，添加到 Tasks [清单名]
2. 为每个任务推荐完成时间，添加到 Calendar
3. 在 Drive 创建项目文档模板
```

## 邮件 × 日历集成

### 邮件转日程
```
扫描最近邮件，找出所有包含以下信息的内容：
- 明确的会议邀请
- 截止日期约定
- 需要我参与的事项
整理成清单，询问我哪些需要加入 Calendar 或 Tasks。
```

### 日程汇总邮件
```
查看明天的日历，生成一封日程提醒邮件草稿，
发给我自己（[我的邮箱]），格式：时间 + 事件 + 地点。
```

## 个人生活场景 (移动)

### 周末规划助手 (Calendar + Tasks)
```
本周末我有哪些日程和未完成任务？
帮我规划一个合理的周末安排，在工作事项和休闲之间取得平衡。
```

### 旅行准备清单 (Calendar + Tasks + Drive)
```
我 [日期] 有一次去 [目的地] 的行程（见日历事件）。
请：
1. 生成一份旅行准备 checklist，添加到 Tasks
2. 在 Drive 创建一个行程文档
3. 提醒我出发前 [N] 天需要完成的准备事项
```

### 每日收尾 (Tasks + Calendar)
```
现在是今天结束，帮我做收尾：
① 列出今天未完成的任务
② 确认明天有哪些重要安排
③ 建议我把哪些任务顺延或重新优先级排序
```
