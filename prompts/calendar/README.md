# Google Calendar Prompts

## 查看与规划

### 今日日程
```
查看我今天所有日历事件，按时间排列，标注地点和参会人（如有）。
```

### 本周概览 + 空闲时间
```
展示本周（周一到周日）完整日程，找出工作时间（9:00–18:00）内空闲超过 1.5 小时的时间窗口。
```

### 下周预览
```
帮我预览下周日程，统计会议总时长，找出最繁忙的一天和最轻松的一天。
```

## 创建与编辑

### 创建事件
```
帮我新建日历事件：
标题：[xxx]
时间：[日期] [开始]–[结束]
地点/链接：[xxx]
参会人：[邮箱列表]
提醒：提前 [15/30/60] 分钟
备注：[xxx]
```

### 修改事件
```
把 [事件名] 从 [原时间] 改到 [新时间]，若有冲突先告诉我。
```

### 删除事件
```
取消 [日期] 的 [事件名]，如有其他参会人，提醒我是否需要发取消通知。
```

## 移动场景

### 今天剩余日程
```
今天剩余时间还有什么日程？用两行字告诉我。
```

### 明早准备提醒
```
明天第一个事件是什么？告诉我时间、地点和需要提前准备的事项。
```
