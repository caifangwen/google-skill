# Google Sheets Prompts

## 数据读取

### 读取表格数据
```
读取 [Sheet 链接] 的 [Sheet 名称] 工作表，
总结数据概况：行数、列名、关键统计值（均值/最大/最小）。
```

### 数据查询
```
在 [Sheet 链接] 中找出所有 [条件，如"状态=未完成"] 的行，列出结果。
```

## 数据写入与更新

### 更新单元格
```
把 [Sheet 链接] 中 [Sheet 名] 的 [行/列条件] 对应的 [字段] 更新为 [新值]。
```

### 追加新行
```
向 [Sheet 链接] 的 [Sheet 名] 追加一行数据：
[字段 1]: [值 1]
[字段 2]: [值 2]
...
```

## 分析与报告

### 数据分析
```
读取 [Sheet 链接]，分析 [指标/趋势]，用文字描述关键发现，
并建议 2–3 个后续行动。
```

### 自动周报表格
```
每周五帮我在 [Sheet 链接] 新增一行，记录本周：
完成事项数、会议次数、待办遗留数。
数据来源：Calendar 和 Tasks。
```
