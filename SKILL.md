# Google Family Bucket Skill

Google 全家桶 Claude Prompt 完全手册 - 60+ 提示词覆盖 Gmail、Calendar、Drive、Docs、Sheets、Slides、Tasks、Forms、Chat、Contacts

## Coverage

- 📧 Gmail - 邮件管理
- 📅 Google Calendar - 日历管理
- 💾 Google Drive - 云存储
- 📝 Google Docs - 文档
- 📊 Google Sheets - 表格
- 🎨 Google Slides - 演示文稿
- ✅ Google Tasks - 任务管理
- 📋 Google Forms - 表单
- 💬 Google Chat - 即时消息
- 👥 Google Contacts - 联系人
- 🔗 Integration - 跨工具集成

## Usage

```bash
# View all prompts
ls prompts/

# View Gmail prompts
ls prompts/gmail/

# View integration workflows
ls prompts/integration/
```
